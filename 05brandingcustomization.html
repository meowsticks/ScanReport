05 · Side spine variants & tier label preview

Two things live here. Side spine ships in v1.0.15 with the right edge as default — a thin vertical band with the company name + stacked page indicator. Tier label customization previews v1.0.16 — changing CLEAR/CAUTION/DO NOT CORE to user-defined wording, deferred because it touches data model + every scan card render.

Logo: each letterhead references assets/logo.png. Commit the full Aggarwal Kamikazes logo there and it renders; until then the brown circular fallback shows. Operator appears as a literal {operator} token — filled per-report at scan time.

What you do not see here: a brand-color picker. Color inside a scan report is semantic — black = top bar, blue = bottom bar, red = conduit / no-scan zone. The letterhead and footer chrome stay neutral so the scan grammar reads clearly.
