v1.0.15 scope — PDF polish + branding (4 features)

  • Branded letterhead — user-uploadable logo + company name + tagline + project no. + address + phone + email + operator, fixed at top of every page. Chrome stays neutral black — no colored accent. Absorbs the standalone "auto project numbers" feature.
  • Side spine — thin vertical band on the right edge (default; flippable to left) with company name running top-to-bottom AND a stacked page indicator at the top (current page / divider / total). Replaces a bottom footer band — pagination lives on the spine, vertical space goes to photos.
  • Page-break-safe scan cards (taller photos) — auto-keep-whole + visible page-break line in preview for drag-reorder fine-tuning. Photos given ~150px height (was 80) since the footer band is gone. Engineers stop losing context mid-card.
  • DRAFT watermark — diagonal overlay on un-issued PDFs. Already in the PDF rendering layer; near-zero extra cost.

Color reserved. The scan body uses black for top-bar markings, blue for bottom-bar, red for conduits and no-scan zones. The chrome (letterhead, spine, project no.) is kept neutral so it doesn't compete with the scan grammar.

Brand assets needed before ship: commit the full Aggarwal Kamikazes logo (mascot + wordmark, as supplied) to docs/v1015-mockups/assets/logo.png. One file is enough; renderer scales it for the letterhead slot.
