# AK ScanReport — Getting Started (v1.0.15)

Quick guide to install and build your first scan report. Two-minute read.

---

## 1. Install

1. Download the installer: **AK.ScanReport.Setup.exe**
   https://github.com/meowsticks/ScanReport/releases/latest
2. Double-click it. Windows will show a blue **"Windows protected your PC"** box
   (it's our own app, just not code-signed yet) → click **More info → Run anyway**.
3. Choose where to install → Finish. You'll get a Desktop + Start-menu shortcut.

**It keeps itself up to date** — when a new version is published, the app pulls it
automatically next launch. No reinstalling.

> Windows only. The version number shows at the very bottom of the app
> (e.g. `· v1.0.15`) so you always know you're current.

---

## 2. Build a report — the flow

1. **Project info** — client, site address, scan area, date.
2. **Tier** (top buttons): **Quick Mark** (1-page field), **Standard** (engineer
   review), **Full** (regulatory). The tier sets sensible defaults; you can flip
   any section on/off yourself.
3. **Targets / Findings** — what you located (rebar, PT, conduit, etc.).
4. **Scan locations & photos** — add a photo, hit **🖊 Annotate** to mark it up
   (arrows, text, freehand) in your color-key colors.
5. **Core verdicts** — Safe / Caution / No-go per core, with clearance + crew
   instructions.
6. **Save / Send PDF** — your deliverable. Email it to the engineer/client.
7. **💾 Save** — a backup file you can reopen and edit later.

---

## 3. New / handy in this version

- **Cleaner report** — professional letterhead, consistent layout, clean white
  PDF pages (the old black border on saved PDFs is gone).
- **Editable Markup Color Key** — click a colour swatch to recolour it or change
  its meaning to match a client/region standard. The annotation tools draw in
  those same colours.
- **Side panel (left)** — show/hide and reorder report sections with checkboxes +
  ▲▼ arrows, and click any name to jump to it. No digging into Setup.
- **"At a glance" (right)** — Safe / Caution / No-go counts; click one to jump
  straight to those core cards.
- **Editable footer** — set your own sign-off line and the small print line (or
  clear them).
- **CAD-style drawing page** — optional one-page engineered drawing sheet
  (letterhead, drawing, notes, title block).
- **Notepad** (bottom-right) — quick scratch notes; saved on your device, never
  printed.
- **📁 Show file** — jump to where your saved report file lives on disk.
- **"FOR REVIEW" watermark** — printed until a report is approved.

---

## 4. Saving & sending

- **Save / Send PDF** → the PDF you email out. PDFs open in any viewer (Adobe,
  Edge, phone) — that's what engineers want.
- **💾 Save** → a backup `.akscan` file. Reopen it any time with **📂 Load** or
  File → Open. New saves never overwrite an old one.
- Your work also **auto-saves on the device** as you go (watch the green
  **✓ Saved** tag at the top).
