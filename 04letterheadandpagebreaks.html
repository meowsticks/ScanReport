★ 04 · Letterhead, spine & page-break preview

The heart of v1.0.15. Fixed letterhead at the top of every page (logo + company + project no. + operator + contact), side spine on the right edge with company name + page indicator, and scan cards that never split across page breaks. Photos given more vertical room since the bottom footer band is gone — pagination moved to the spine.

Color note: chrome (letterhead rule, project no.) is neutral black. The scan content reserves color semantically — black for top bar, blue for bottom bar, red for conduits and no-scan zones — so the chrome must not introduce a competing accent.

Side spine carries pagination. Page number stacked above a divider line, with the total below — e.g., 1 / — / 2 on page one, 2 / — / 2 on page two. Company name fills the rest of the spine. No footer band; cards get the freed vertical space.

Logo: commit the full Aggarwal Kamikazes logo at docs/v1015-mockups/assets/logo.png (lowercase, named exactly logo.png). All letterhead slots reference that path — once the file is in place, refresh and the actual mark renders. Until then, the brown circular fallback shows.

Operator shown as literal {operator} token — per-report slot, filled by whoever creates the scan.
